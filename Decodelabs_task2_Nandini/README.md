# 6-Axis Robotic Arm Simulator Using Inverse Kinematics

## 📌 Project Overview

The **6-Axis Robotic Arm Simulator Using Inverse Kinematics** is a Python-based simulation project that demonstrates how a robotic arm can move its end effector to a target position by calculating the required joint angles using inverse kinematics. The project provides a simple graphical visualization of robotic arm movement without requiring any physical hardware.

---

## 🎯 Objectives

- Design a virtual robotic arm.
- Implement inverse kinematics calculations.
- Calculate joint angles for a target position.
- Simulate robotic arm movement.
- Visualize the arm using Python and Matplotlib.
- Understand the basics of robotic manipulation and automation.

## ⚙ Installation

```bash

```

### Install required libraries

```bash

```

## ▶ Run the Project

Execute the Python file:

