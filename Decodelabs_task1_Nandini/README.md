# Automated Quality Inspection System Using Computer Vision

## Project Overview

The Automated Quality Inspection System is a Computer Vision project developed using Python and OpenCV. The system analyzes product images, detects contours and edges, and classifies products as either GOOD PRODUCT or DEFECTIVE PRODUCT.

This project demonstrates the use of image processing techniques for industrial quality control and defect detection.

## Features

* Image Acquisition
* Grayscale Conversion
* Gaussian Blur Filtering
* Edge Detection using Canny Algorithm
* Contour Detection
* Product Classification
* Automated Quality Inspection

## Technologies Used

* Python
* OpenCV
* Matplotlib
* Jupyter Notebook
